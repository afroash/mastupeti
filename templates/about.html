About This Project

Welcome,

This project is a fully functional web application developed as a learning project. The primary goal of this project is to understand and apply the basics of the Go programming language and to gain hands-on experience in deploying applications to the AWS cloud.

Features of this application include:

  • A contact form that sends emails using SMTP.
  • Implementation of the Gin web framework for routing and handling requests.
  • Use of the GORM library for interacting with a PostgreSQL database.
  • Authentication using JWT tokens.
  • Use of the AWS SDK for sending emails.
  • Use of the AWS SDK for interacting with S3.
  • Deployment on AWS, showcasing skills in cloud infrastructure.

This project is listed on my CV as a demonstration of my ability to learn new technologies and apply them in a practical, real-world scenario. It reflects my commitment to continuous learning and my capability to manage and deploy applications in a cloud environment.
